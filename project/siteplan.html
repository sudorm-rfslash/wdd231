Site Plan for My Web Project

Site Name

Name: CineView

Reason for Selection: I wanted to create a website to 'view' information about 'cinema' or movies. 'Cine' is also spanish for cinema.

Site Purpose

I want my website to show basic movie information to the customer. Things like title and poster being the highest priority. Then when you click the poster you can have a modal show additional information.

Scenarios

Examples of questions a site visitor might have:

Where do you pull your website information from?:
  • I plan to use The Movie DataBase (TMDB) to get my movie info.
Why should I use this website isntead of XYZ movie site?
  • I want my site to be designed clean and clutter free. There shouldn't be any distractions like ads or other unnecessary information.

Color Schema

Primary Color: #005AA3 (used in headers/footers and accents)

Secondary Color: #FFFFFF (used in body text)

Typography

Font Choice: I will use Roboto because I think it looks nice. I will use headings or subheading to make things like labels stand out. I can also expirement with bold text using the 'strong' tag.
